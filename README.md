# stack-visualizer
Stack Visualization using array

## 🥞 Stack Visualizer
A modern, interactive web application to visualize Stack Data Structure operations (Push, Pop, Peek) in real-time.
## ✨ Features

* Visual Stacking: Watch elements drop into the stack with smooth CSS animations.
* Dynamic Feedback: Real-time messages for successful operations and errors like Stack Overflow (Full) and Stack Underflow (Empty).
* Top Highlight: The top element of the stack is automatically highlighted with a distinct glow and color.
* Responsive Design: A sleek, glassmorphism-inspired dark mode layout that works on different screen sizes. 

## 🛠️ Tech Stack

* Frontend: HTML5, CSS3 (Gradients, Flexbox, Animations)
* Logic: JavaScript (DOM Manipulation, Array implementation)
